# Tavily Search API Utilities

**Book Chapter:** [Web Scraping](https://leanpub.com/racket-ai/read) — *Practical Artificial Intelligence Development With Racket* (free to read online).

This project provides utilities for using the Tavily Search API in Racket. Tavily is specifically optimized for LLMs and AI agents to gather factual information efficiently.

## Run

Ensure you have your Tavily API key set in your environment before running.

    racket tavily_search.rkt
